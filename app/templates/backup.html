Recovery Notes / 恢复说明
Restore replaces the target database content. Only run restore on a new server or after confirming the current database can be overwritten. / 恢复会覆盖目标数据库内容，只应在新服务器或确认可覆盖当前数据库时执行。
Full guide: docs/Backup_and_Restore.md
Backup package includes MySQL dump, .env, Docker config, Git version and restore notes. / 备份包包含数据库、环境配置、Docker 配置、Git 版本和恢复说明。
